Custom Short URLs in WordPress

I mentioned in my earlier post that if you create a Short URL of a page on this site using the Short URL Safari Extension, you’ll be automatically delivered an ihin.es URL. A number of people have asked me how this works. Here’s how.

  1. I created a bit.ly account.
  2. I installed the “Simple URL Shortener” WordPress plugin, which can be found here.
  3. I entered the following code, which calls the “Simple URL Shortener” plugin, within my site’s header:
    <?php if (function_exists('simple_url_shortener')) : ?> <link rel="shortlink" href="<?php echo simple_url_shortener('','service=bit.ly+key&apikey= APIKEY&login=USERNAME&cache=no'); ?>" /> <?php endif; ?> <link rel="canonical" href="<?php the_permalink() ?>"/>

    Obviously you’ll need to replace APIKEY and USERNAME with your bit.ly username and API key.

  4. That’s it.

This has the effect of creating an element in your site’s header identifying your custom bit.ly URL as the specified “shorturl,” which a variety of URL shorteners recognize and use. It won’t work in every instance, but consider it another tool in your arsenal of awesome.

Note: To create short URLs with a custom domain name, simply find and register said domain name using Domai.nr and configure it as a bit.ly pro URL here.
